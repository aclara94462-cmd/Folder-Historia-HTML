O que é HTML

HTML (Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web.

Principais versões do HTML

  • HTML 2.0 (1995)
  • HTML 3.2 (1997)
  • HTML 4.01 (1999)
  • HTML 1.0 (2000)
  • HTML 5 (2014)
